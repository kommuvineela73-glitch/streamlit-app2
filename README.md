# Business Management System

## Project Description
This is a Business Management System built using Streamlit.  
It helps manage business transactions, inventory, and reports.

## Features
- User Login and Registration
- Dashboard for Sales and Expenses
- Inventory Management
- Transaction Tracking
- Excel Report Generation
- PDF Report Generation
- Admin Dashboard

## Technologies Used
- Python
- Streamlit
- Pandas
- FPDF

## How to Run the Project

Install dependencies:

pip install -r requirements.txt

Run the application:

streamlit run app.py
